Gesture Guide

This system detects different hand gestures.

  • 👍 Thumbs Up → Only thumb
  • ✌ Peace → Index + Middle
  • ☝ Pointing → Index finger
  • 3 Fingers → Index + Middle + Ring
  • 4 Fingers → All except thumb
  • 🖐 Open Hand → All fingers
  • ✊ Fist → No fingers
  • 🤙 Call Me → Thumb + Pinky
  • 👌 OK → Thumb + Index
